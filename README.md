# Healthcare ROI Landing Page

A premium, high-performance landing page for the Healthcare ROI Calculator, built with Next.js 14, Tailwind CSS, and Framer Motion.

## Features
- **Cinematic Motion**: Smooth entrance animations and parallax map background.
- **Live Search**: Real-time company filtering with keyboard navigation.
- **Responsive Design**: Mobile-first approach scaling to large desktops.
- **Performance**: Optimized assets and zero-layout-shift architecture.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + CSS Modules (Glassmorphism)
- **Animation**: Framer Motion
- **Language**: TypeScript

## Project Structure
- `app/`: Routes and global styles.
- `components/`: Reusable UI components (`Hero`, `MapBackground`, etc.).
- `content/`: Static content files (`facts.json`).
- `lib/`: Utility functions and mock data.
- `public/`: Static assets (`us-map.svg`).
